# HOPFIELD NEURAL NETWORK

This is a mini-project meant to visualize the functionality of Classical Hopfield Networks. The learning method we used is a very simple in this case: Hebbian Learning. It is very easy to setup and the explanation is right below this section. Hope you enjoy this mini-project!

### How to setup?

Create a .venv if necessary. Python version does not matter but preferrably >=3.11.
1. Install the packages in requirements.txt. "pip install -r requirements.txt"
2. Run hopfield_mnist.py. "python hopfield_mnist.py"

You can change the patterns you want it to learn in the main function of the file. The constant variables are changeable based on preferences.

Hope you enjoy!
